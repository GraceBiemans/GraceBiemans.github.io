Pet Groomers

Goal was to solve a synchronization problem presented. There are multiple kinds of pets, a set number of grooming stations, and a set of rules we would have to enforce.

Rules:
1. Cats and dogs cannot be groomed at the same time
2. Only one pet can use a grooming station at a time
3. Pets cannot be forced to wait indefinitely / be neglected
4. Must be an efficient solution


Use of C and POSIX Threads
